# Crow_ Project

Crow_ is a front-end development project created as part of the Front-End Development course at Systems Analysis and Development. It is designed to showcase various front-end development techniques and features.


## Technologies Used

- **HTML**: Used for the structure and markup of the web pages.
- **CSS**: Used for styling the web pages and creating animations.
- **JavaScript**: Used for adding interactivity and dynamic behavior to the website.
- **External Libraries**: Utilizes external libraries like Font Awesome for icons, ParticlesJS and AOS for animation.
